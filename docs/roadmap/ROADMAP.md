# Roadmap

This document describes the planned evolution of **RAG-PLATAFORM**.

The roadmap reflects the natural growth of the platform as an **Omnichannel AI Gateway with RAG capabilities**.

---

# Version Strategy

The project follows **Semantic Versioning**.

Format:

MAJOR.MINOR.PATCH

Example:

v1.2.3

- MAJOR → breaking changes
- MINOR → new features
- PATCH → bug fixes

---

# v0.1.0 — Initial Platform Release

Status: Released

Features included:

- NestJS backend
- Next.js dashboard
- Omnichannel gateway
- Telegram connector
- Email (dev provider)
- RAG gateway integration
- PostgreSQL persistence (port 5433)
- Observability stack
- Docker environment
- CI/CD pipelines
- Operational dashboard

RAG-PLATAFORM aims to become a **full AI communication orchestration platform**, capable of:

- receiving requests from any channel
- routing to intelligent agents
